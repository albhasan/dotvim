# dotvim
vim configuration files

Initialization:

    git clone https://github.com/albhasan/dotvim.git

Create symlinks:

    ln -s ~/dotvim/vimrc                  ~/.vimrc
    ln -s ~/dotvim                        ~/.vim                          
    ln -s ~/dotvim/tmux-config/.tmux.conf ~/.tmux.conf

Resources:

- The Tao of tmux https://leanpub.com/the-tao-of-tmux/
